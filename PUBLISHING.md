# PUBLISHING

## Pre-publish flow
1. `./.github/skills-config/ebook-build/invoke-build.ps1` を実行する
2. 以下の成果物を確認する
   - `ebook-output/github-copilot-adoption-framework.epub`
   - `ebook-output/github-copilot-adoption-framework.pdf`
   - `ebook-output/github-copilot-adoption-framework-kdp-registration.md`
3. `README.md` と `docs/index.md` のリンク導線を確認する
4. 図版・`docs/images/`・KDP 登録情報の参照に不整合がないか確認する
5. GitHub Pages を確認して公開する
